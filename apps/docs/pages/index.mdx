# Prova Documentation

**Prova** is a behavior attestation layer that issues verifiable, signed, immutable receipts of every AI agent action on Solana, built on top of the [Solana Attestation Service (SAS)](https://attest.solana.com).

## What is Prova?

When an AI agent executes an action — a transaction, a tool call, a decision — Prova issues a **cryptographic attestation** on Solana. This attestation is:

- **Signed** by the agent's Ed25519 key
- **Immutable** — stored on Solana, cannot be altered
- **Verifiable** — anyone can verify without trusting Prova
- **Composable** — built on SAS, standard schemas
